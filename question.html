Question-1: What are the different ways to select an element in the DOM?

There are few ways to select an element in DOM.They are getElementById("id"),getElementsByClassName("class"),getElementsByTagName("tag"),querySelector("selector"),querySelectorAll("selector")

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The differences between innerHTML, innerText, and textContent are:
innerHTML – Gets or sets the HTML content inside an element, including tags.
innerText – Gets or sets only the visible text inside an element.
textContent – Gets or sets all text inside an element, including hidden elements.

Question-3: What is event delegation in the DOM?

Event delegation is a way where a parent element handles events for its child elements using event bubbling. Instead of adding event listeners to multiple child elements, you add one listener to the parent and use event.target to identify which child triggered the event.

Question-4: What is event bubbling in the DOM?

Event bubbling is a porcess when an event starts on the element that was clicked and then moves up through its parent elements, one by one, until it reaches the "html" tag.

Question-5: How do you create, add, and remove elements using JavaScript?

For create new element we use "document.createElement()"".
For adding element we use appendChild() or append() to add the element inside a parent.
For removing an element we use remove() to delete an element.
